Lección 1

Selección de Elementos

Aprende los diferentes métodos para seleccionar elementos del DOM y trabajar con ellos en JavaScript.

🎯 Puntos Clave

Métodos de selección
Selectores CSS
Colecciones de elementos

Teoría

Métodos de Selección

getElementById() Selecciona por ID único
querySelector() Selecciona con CSS selector
querySelectorAll() Selecciona múltiples elementos
getElementsByClassName() Selecciona por clase

Selectores CSS

#id Por ID
.clase Por clase
elemento Por etiqueta
[atributo] Por atributo

Ejemplo Práctico

Selección Básica 
// Selección por ID
const titulo = document.getElementById('titulo');
console.log('Elemento por ID:', titulo);

// Selección por clase
const parrafos = document.getElementsByClassName('parrafo');
console.log('Elementos por clase:', parrafos);

// Selección por etiqueta
const divs = document.getElementsByTagName('div');
console.log('Elementos div:', divs);

// Selección con querySelector
const primerParrafo = document.querySelector('.parrafo');
console.log('Primer párrafo:', primerParrafo);

// Selección múltiple con querySelectorAll
const todosParrafos = document.querySelectorAll('.parrafo');
console.log('Todos los párrafos:', todosParrafos);

// Selección por atributo
const enlaces = document.querySelectorAll('a[href]');
console.log('Enlaces con href:', enlaces);
Selección Avanzada 
// Selectores complejos
const elementos = document.querySelectorAll('div.contenido > p:first-child');
console.log('Párrafos hijos directos:', elementos);

// Selección por atributos específicos
const inputs = document.querySelectorAll('input[type="text"]');
console.log('Inputs de texto:', inputs);

// Selección por pseudo-clases
const hoverElements = document.querySelectorAll('button:hover');
console.log('Botones en hover:', hoverElements);

// Navegación en el DOM
const contenedor = document.querySelector('.contenedor');
const hijos = contenedor.children;
const padre = contenedor.parentElement;
const hermanos = contenedor.nextElementSibling;

console.log('Hijos del contenedor:', hijos);
console.log('Elemento padre:', padre);
console.log('Hermano siguiente:', hermanos);

// Verificar si existe un elemento
const existe = document.getElementById('no-existe');
if (existe) {
    console.log('El elemento existe');
} else {
    console.log('El elemento no existe');
}

    this.style.border = '2px solid blue';
});

    this.style.border = '1px solid gray';
});

Event Delegation 
// Crear lista dinámica
const lista = document.createElement('ul');
lista.id = 'miLista';
document.body.appendChild(lista);

// Agregar algunos elementos
for (let i = 1; i <= 3; i++) {
    const li = document.createElement('li');
    li.textContent = `Elemento ${i}`;
    li.className = 'item-lista';
    li.style.cursor = 'pointer';
    li.style.padding = '5px';
    li.style.margin = '2px';
    li.style.border = '1px solid #ccc';
    lista.appendChild(li);
}

// Event delegation - un solo listener para todos los elementos
lista.addEventListener('click', function(event) {
    // Verificar si el elemento clickeado es un li
    if (event.target.classList.contains('item-lista')) {
        console.log('Elemento clickeado:', event.target.textContent);
        event.target.style.backgroundColor = 'yellow';
    }
});

// Evento de mouseover
lista.addEventListener('mouseover', function(event) {
    if (event.target.classList.contains('item-lista')) {
        event.target.style.backgroundColor = 'lightgreen';
    }
});

// Evento de mouseout
lista.addEventListener('mouseout', function(event) {
    if (event.target.classList.contains('item-lista')) {
        event.target.style.backgroundColor = '';
    }
});

console.log('Event delegation configurado. Haz clic en los elementos de la lista.');

Calculadora Interactiva

Principiante

Crea una calculadora básica que permita sumar, restar, multiplicar y dividir.

Requisitos:

  • Crear interfaz con botones y display
  • Manejar eventos de click
  • Mostrar resultados en tiempo real
  • Validar división por cero
// Crear interfaz de calculadora
const calculadora = document.createElement('div');
calculadora.style.cssText = `
    width: 300px;
    margin: 20px auto;
    padding: 20px;
    border: 2px solid #333;
    border-radius: 10px;
    background: #f5f5f5;
`;

const display = document.createElement('input');
display.type = 'text';
display.id = 'display';
display.style.cssText = `
    width: 100%;
    height: 50px;
    font-size: 24px;
    text-align: right;
    margin-bottom: 10px;
    border: 1px solid #ccc;
    border-radius: 5px;
`;

calculadora.appendChild(display);

// Crear botones
const botones = [
    ['C', 'CE', '/', '*'],
    ['7', '8', '9', '-'],
    ['4', '5', '6', '+'],
    ['1', '2', '3', '='],
    ['0', '.', '', '']
];

let operacion = '';
let numero1 = '';
let numero2 = '';

botones.forEach(fila => {
    const filaDiv = document.createElement('div');
    filaDiv.style.display = 'flex';
    filaDiv.style.gap = '5px';
    filaDiv.style.marginBottom = '5px';
    
    fila.forEach(texto => {
        if (texto) {
            const boton = document.createElement('button');
            boton.textContent = texto;
            boton.style.cssText = `
                flex: 1;
                height: 50px;
                font-size: 18px;
                border: 1px solid #ccc;
                border-radius: 5px;
                cursor: pointer;
                background: white;
            `;
            
            boton.addEventListener('click', () => manejarClick(texto));
            filaDiv.appendChild(boton);
        } else {
            const espacio = document.createElement('div');
            espacio.style.flex = '1';
            filaDiv.appendChild(espacio);
        }
    });
    
    calculadora.appendChild(filaDiv);
});

function manejarClick(valor) {
    if (valor >= '0' && valor <= '9' || valor === '.') {
        if (!operacion) {
            numero1 += valor;
            display.value = numero1;
        } else {
            numero2 += valor;
            display.value = numero2;
        }
    } else if (['+', '-', '*', '/'].includes(valor)) {
        operacion = valor;
    } else if (valor === '=') {
        calcular();
    } else if (valor === 'C') {
        limpiar();
    } else if (valor === 'CE') {
        display.value = '';
    }
}

function calcular() {
    const n1 = parseFloat(numero1);
    const n2 = parseFloat(numero2);
    let resultado = 0;
    
    switch (operacion) {
        case '+': resultado = n1 + n2; break;
        case '-': resultado = n1 - n2; break;
        case '*': resultado = n1 * n2; break;
        case '/': 
            if (n2 === 0) {
                display.value = 'Error: División por cero';
                return;
            }
            resultado = n1 / n2; 
            break;
    }
    
    display.value = resultado;
    numero1 = resultado.toString();
    numero2 = '';
    operacion = '';
}

function limpiar() {
    numero1 = '';
    numero2 = '';
    operacion = '';
    display.value = '';
}

document.body.appendChild(calculadora);

// Crear interfaz de lista de tareas
const todoApp = document.createElement('div');
todoApp.style.cssText = `
    max-width: 600px;
    margin: 20px auto;
    padding: 20px;

`;

// Input para nueva tarea
const inputContainer = document.createElement('div');
inputContainer.style.cssText = `
    display: flex;
    gap: 10px;
    margin-bottom: 20px;
`;

const inputTarea = document.createElement('input');
inputTarea.type = 'text';
inputTarea.placeholder = 'Nueva tarea...';
inputTarea.style.cssText = `
    flex: 1;

`;

const btnAgregar = document.createElement('button');
btnAgregar.textContent = 'Agregar';

// Filtros
const filtros = document.createElement('div');
filtros.style.cssText = `

const btnTodos = document.createElement('button');
btnTodos.textContent = 'Todas';

btnPendientes.textContent = 'Pendientes';
btnPendientes.className = 'filtro-btn';

const btnCompletadas = document.createElement('button');
btnCompletadas.textContent = 'Completadas';

[filtros, btnTodos, btnPendientes, btnCompletadas].forEach(el => {
    if (el !== filtros) {
        el.style.cssText = `
            padding: 5px 15px;
            border: 1px solid #ccc;
            background: white;
            cursor: pointer;
            border-radius: 5px;
        `;
        filtros.appendChild(el);
    }
});

// Lista de tareas
const listaTareas = document.createElement('ul');
listaTareas.id = 'lista-tareas';
listaTareas.style.cssText = `
    list-style: none;
    padding: 0;
    margin: 0;
`;

// Contador
const contador = document.createElement('div');
contador.id = 'contador';
contador.style.cssText = `
    margin-top: 20px;
    padding: 10px;
    background: #f8f9fa;
    border-radius: 5px;
    text-align: center;
`;

// Estado de la aplicación
let tareas = JSON.parse(localStorage.getItem('tareas')) || [];
let filtroActual = 'todas';

// Funciones
function renderizarTareas() {
    listaTareas.innerHTML = '';
    
    const tareasFiltradas = tareas.filter(tarea => {
        if (filtroActual === 'pendientes') return !tarea.completada;
        if (filtroActual === 'completadas') return tarea.completada;
        return true;
    });
    
    tareasFiltradas.forEach((tarea, index) => {
        const li = document.createElement('li');
        li.style.cssText = `
            display: flex;
            align-items: center;
            padding: 10px;
            border: 1px solid #eee;
            margin-bottom: 5px;
            border-radius: 5px;
            background: ${tarea.completada ? '#d4edda' : 'white'};
        `;
        
        const checkbox = document.createElement('input');
        checkbox.type = 'checkbox';
        checkbox.checked = tarea.completada;
        checkbox.addEventListener('change', () => toggleTarea(tarea.id));
        
        const texto = document.createElement('span');
        texto.textContent = tarea.texto;
        texto.style.cssText = `
            flex: 1;
            margin: 0 10px;
            text-decoration: ${tarea.completada ? 'line-through' : 'none'};
            color: ${tarea.completada ? '#6c757d' : '#333'};
        `;
        
        const btnEliminar = document.createElement('button');
        btnEliminar.textContent = 'Eliminar';
        btnEliminar.style.cssText = `
            padding: 5px 10px;
            background: #dc3545;
            color: white;
            border: none;
            border-radius: 3px;
            cursor: pointer;
        `;
        btnEliminar.addEventListener('click', () => eliminarTarea(tarea.id));
        
        li.appendChild(checkbox);
        li.appendChild(texto);
        li.appendChild(btnEliminar);
        listaTareas.appendChild(li);
    });
    
    actualizarContador();
}

function agregarTarea() {
    const texto = inputTarea.value.trim();
    if (texto) {
        const nuevaTarea = {
            id: Date.now(),
            texto: texto,
            completada: false
        };
        tareas.push(nuevaTarea);
        inputTarea.value = '';
        guardarTareas();
        renderizarTareas();
    }
}

function toggleTarea(id) {
    const tarea = tareas.find(t => t.id === id);
    if (tarea) {
        tarea.completada = !tarea.completada;
        guardarTareas();
        renderizarTareas();
    }
}

function eliminarTarea(id) {
    tareas = tareas.filter(t => t.id !== id);
    guardarTareas();
    renderizarTareas();
}

function setFiltro(filtro) {
    filtroActual = filtro;
    document.querySelectorAll('.filtro-btn').forEach(btn => {
        btn.classList.remove('active');
        btn.style.background = 'white';
    });
    event.target.classList.add('active');
    event.target.style.background = '#007bff';
    event.target.style.color = 'white';
    renderizarTareas();
}

function actualizarContador() {
    const total = tareas.length;
    const completadas = tareas.filter(t => t.completada).length;
    const pendientes = total - completadas;
    
    contador.innerHTML = `
        Total: ${total} | Completadas: ${completadas} | Pendientes: ${pendientes}
    `;
}

function guardarTareas() {
    localStorage.setItem('tareas', JSON.stringify(tareas));
}

// Event listeners
btnAgregar.addEventListener('click', agregarTarea);
inputTarea.addEventListener('keypress', (e) => {
    if (e.key === 'Enter') agregarTarea();
});

btnTodos.addEventListener('click', () => setFiltro('todas'));
btnPendientes.addEventListener('click', () => setFiltro('pendientes'));
btnCompletadas.addEventListener('click', () => setFiltro('completadas'));

// Montar la aplicación
inputContainer.appendChild(inputTarea);
inputContainer.appendChild(btnAgregar);

todoApp.appendChild(inputContainer);
todoApp.appendChild(filtros);
todoApp.appendChild(listaTareas);
todoApp.appendChild(contador);

document.body.appendChild(todoApp);

// Renderizar inicial
renderizarTareas();

Formulario de Contacto Interactivo

// Formulario de contacto con validación
const form = document.getElementById('contactForm');
const inputs = form.querySelectorAll('input, textarea');

// Validación en tiempo real
inputs.forEach(input => {
    input.addEventListener('blur', function() {
        validarCampo(this);
    });
    
    input.addEventListener('input', function() {
        if (this.classList.contains('error')) {
            validarCampo(this);
        }
    });
});

function validarCampo(campo) {
    const valor = campo.value.trim();
    let esValido = true;
    let mensaje = '';

    if (campo.hasAttribute('required') && !valor) {
        esValido = false;
        mensaje = 'Este campo es obligatorio';
    } else if (campo.type === 'email' && valor) {
        const emailRegex = /^[^\s@]+@[^\s@]+\.[^\s@]+$/;
        if (!emailRegex.test(valor)) {
            esValido = false;
            mensaje = 'Ingresa un email válido';
        }
    }

    if (esValido) {
        campo.classList.remove('error');
        campo.setCustomValidity('');
        ocultarError(campo);
    } else {
        campo.classList.add('error');
        campo.setCustomValidity(mensaje);
        mostrarError(campo, mensaje);
    }
}

// Envío del formulario
form.addEventListener('submit', function(e) {
    e.preventDefault();
    
    const formData = new FormData(this);
    const datos = Object.fromEntries(formData);
    
    console.log('Datos del formulario:', datos);
    alert('¡Formulario enviado correctamente!');
    form.reset();
    limpiarErrores();
});

function mostrarError(campo, mensaje) {
    let errorDiv = campo.parentNode.querySelector('.error-message');
    if (!errorDiv) {
        errorDiv = document.createElement('div');
        errorDiv.className = 'error-message';
        campo.parentNode.appendChild(errorDiv);
    }
    errorDiv.textContent = mensaje;
}

function ocultarError(campo) {
    const errorDiv = campo.parentNode.querySelector('.error-message');
    if (errorDiv) {
        errorDiv.remove();
    }
}

function limpiarErrores() {
    inputs.forEach(input => {
        input.classList.remove('error');
        input.setCustomValidity('');
        ocultarError(input);
    });
}

const form = document.getElementById('formulario');

// Validación en tiempo real
inputs.forEach(input => {
    input.addEventListener('blur', function() {
        validarCampo(this);
    });
    
    input.addEventListener('input', function() {
        if (this.classList.contains('error')) {
            validarCampo(this);
        }
    });
});

function validarCampo(campo) {
    const valor = campo.value.trim();
    let esValido = true;
    let mensaje = '';

    // Validaciones específicas
    if (campo.hasAttribute('required') && !valor) {
        esValido = false;
        mensaje = 'Este campo es obligatorio';
    } else if (campo.type === 'email' && valor) {
        const emailRegex = /^[^\s@]+@[^\s@]+\.[^\s@]+$/;
        if (!emailRegex.test(valor)) {
            esValido = false;
            mensaje = 'Ingresa un email válido';
        }
    } else if (campo.type === 'tel' && valor) {
        const telefonoRegex = /^\d{10}$/;
        if (!telefonoRegex.test(valor)) {
            esValido = false;
            mensaje = 'Ingresa un teléfono válido (10 dígitos)';
        }
    }

    // Mostrar/ocultar error
    if (esValido) {
        campo.classList.remove('error');
        campo.setCustomValidity('');
        ocultarError(campo);
    } else {
        campo.classList.add('error');
        campo.setCustomValidity(mensaje);
        mostrarError(campo, mensaje);
    }
}

// Envío del formulario
form.addEventListener('submit', function(e) {
    e.preventDefault();
    
    const formData = new FormData(this);
    const datos = Object.fromEntries(formData);
    
    // Validar todos los campos
    let formularioValido = true;
    inputs.forEach(input => {
        validarCampo(input);
        if (input.classList.contains('error')) {
            formularioValido = false;
        }
    });
    
    if (formularioValido) {
        console.log('Datos del formulario:', datos);
        alert('¡Formulario enviado correctamente!');
        form.reset();
        limpiarErrores();
    } else {
        alert('Por favor, corrige los errores antes de enviar');
    }
});

function mostrarError(campo, mensaje) {
    let errorDiv = campo.parentNode.querySelector('.error-message');
    if (!errorDiv) {
        errorDiv = document.createElement('div');
        errorDiv.className = 'error-message';
        campo.parentNode.appendChild(errorDiv);
    }
    errorDiv.textContent = mensaje;
}

function ocultarError(campo) {
    const errorDiv = campo.parentNode.querySelector('.error-message');
    if (errorDiv) {
        errorDiv.remove();
    }
}

function limpiarErrores() {
    inputs.forEach(input => {
        input.classList.remove('error');
        input.setCustomValidity('');
        ocultarError(input);
    });
}

Gestor de LocalStorage

Intermedio

Requisitos:

  • Guardar y recuperar datos
  • Manejo de objetos con JSON
  • Validación de datos
  • Limpieza y gestión de errores
// Clase para manejar localStorage
class LocalStorageManager {
    constructor(prefix = 'app_') {
        this.prefix = prefix;
    }
    
    // Guardar datos
    setItem(key, value) {
        try {
            const dataToStore = typeof value === 'object' 
                ? JSON.stringify(value) 
                : value;
            localStorage.setItem(this.prefix + key, dataToStore);
            return true;
        } catch (error) {
            console.error('Error guardando en localStorage:', error);
            return false;
        }
    }
    
    // Recuperar datos
    getItem(key, defaultValue = null) {
        try {
            const item = localStorage.getItem(this.prefix + key);
            if (item === null) return defaultValue;
            
            // Intentar parsear como JSON
            try {
                return JSON.parse(item);
            } catch {
                return item; // Devolver como string si no es JSON
            }
        } catch (error) {
            console.error('Error recuperando de localStorage:', error);
            return defaultValue;
        }
    }
    
    // Eliminar elemento
    removeItem(key) {
        try {
            localStorage.removeItem(this.prefix + key);
            return true;
        } catch (error) {
            console.error('Error eliminando de localStorage:', error);
            return false;
        }
    }
    
    // Limpiar todos los elementos con nuestro prefijo
    clear() {
        try {
            const keys = Object.keys(localStorage);
            keys.forEach(key => {
                if (key.startsWith(this.prefix)) {
                    localStorage.removeItem(key);
                }
            });
            return true;
        } catch (error) {
            console.error('Error limpiando localStorage:', error);
            return false;
        }
    }
    
    // Obtener todas las claves
    getAllKeys() {
        const keys = Object.keys(localStorage);
        return keys.filter(key => key.startsWith(this.prefix));
    }
}

×

¿Qué es querySelector()?

Es un método moderno que permite seleccionar elementos usando selectores CSS. Es muy flexible y potente.

Sintaxis:

document.querySelector('selector-css');

Ejemplos:

Características:

  • Devuelve el primer elemento que coincida
  • Usa sintaxis de selectores CSS
  • Muy flexible para selecciones complejas
  • Devuelve null si no encuentra nada

querySelectorAll()

×

¿Qué es querySelectorAll()?

Similar a querySelector(), pero devuelve TODOS los elementos que coincidan con el selector, no solo el primero.

Sintaxis:

document.querySelectorAll('selector-css');

Ejemplo:

// HTML: 
1
2

  • Puede devolver 0, 1 o múltiples elementos
  • Se puede iterar con forEach()
  • No se actualiza automáticamente si cambia el DOM

Animación de Elemento con JavaScript

// HTML: <div id="miElemento">Haz clic para animar</div>

const elemento = document.getElementById('miElemento');
let animando = false;

// Función de animación suave
function animarElemento() {
    if (animando) return;
    animando = true;
    
    let posicion = 0;
    const velocidad = 2;
    
    function mover() {
        posicion += velocidad;
        elemento.style.transform = `translateX(${posicion}px)`;
        
        if (posicion < 200) {
            requestAnimationFrame(mover);
        } else {
            // Volver a la posición original
            elemento.style.transition = 'transform 0.5s ease';
            elemento.style.transform = 'translateX(0)';
            setTimeout(() => {
                elemento.style.transition = '';
                animando = false;
            }, 500);
        }
    }
    
    mover();
}

// Evento de clic
elemento.addEventListener('click', animarElemento);

// Alternar clase para efecto visual
elemento.addEventListener('mouseenter', () => {
    elemento.classList.add('hover-effect');
});

elemento.addEventListener('mouseleave', () => {
    elemento.classList.remove('hover-effect');
});

Ejemplo Práctico

Lista de Tareas Persistente

// HTML: <input id="tareaInput" placeholder="Nueva tarea">
//       <button onclick="agregarTarea()">Agregar</button>
//       <ul id="listaTareas"></ul>

let tareas = JSON.parse(localStorage.getItem('tareas')) || [];

// Cargar tareas al iniciar
cargarTareas();

function agregarTarea() {
    const input = document.getElementById('tareaInput');
    const texto = input.value.trim();
    
    if (texto) {
        const tarea = {
            id: Date.now(),
            texto: texto,
            completada: false
        };
        
        tareas.push(tarea);
        guardarTareas();
        cargarTareas();
        input.value = '';
    }
}

function cargarTareas() {
    const lista = document.getElementById('listaTareas');
    lista.innerHTML = '';
    
    tareas.forEach(tarea => {
        const li = document.createElement('li');
        li.innerHTML = `
            <span class="${tarea.completada ? 'completada' : ''}">
                ${tarea.texto}
            </span>
            <button onclick="toggleTarea(${tarea.id})">
                ${tarea.completada ? 'Deshacer' : 'Completar'}
            </button>
            <button onclick="eliminarTarea(${tarea.id})">Eliminar</button>
        `;
        lista.appendChild(li);
    });
}

function toggleTarea(id) {
    const tarea = tareas.find(t => t.id === id);
    if (tarea) {
        tarea.completada = !tarea.completada;
        guardarTareas();
        cargarTareas();
    }
}

function eliminarTarea(id) {
    tareas = tareas.filter(t => t.id !== id);
    guardarTareas();
    cargarTareas();
}

function guardarTareas() {
    localStorage.setItem('tareas', JSON.stringify(tareas));
}

Evento Click

×

¿Qué es el evento click?

Se dispara cuando el usuario hace clic con el mouse sobre un elemento.

Sintaxis:

elemento.addEventListener('click', function(e) {
    // código aquí
});

Ejemplo:

const boton = document.getElementById('miBoton');

boton.addEventListener('click', function(e) {

});

Características:

  • Se dispara en el momento del clic
  • Incluye información del evento (coordenadas, teclas modificadoras)
  • Funciona con cualquier elemento clickeable
  • Se puede prevenir con preventDefault()

¿Qué es el evento keydown?

Se dispara cuando el usuario presiona una tecla del teclado.

elemento.addEventListener('keydown', function(e) {

Propiedades útiles:

  • e.key - Tecla presionada ('a', 'Enter', 'Escape')

  • e.shiftKey - Si Shift está presionado

addEventListener()

×

¿Qué es addEventListener()?

Método para agregar un manejador de eventos a un elemento del DOM.

Sintaxis:

elemento.addEventListener(tipo, función, opciones);

Ejemplo:

const boton = document.getElementById('miBoton');

// Función de manejo
function manejarClick(e) {
    console.log('¡Clic!', e);
}

// Agregar evento
boton.addEventListener('click', manejarClick);

// Múltiples eventos
boton.addEventListener('mouseenter', function() {
    this.style.backgroundColor = 'blue';
});

boton.addEventListener('mouseleave', function() {
    this.style.backgroundColor = '';
});

Ventajas sobre onclick:

  • Permite múltiples manejadores
  • Mejor control de propagación
  • Fácil de remover con removeEventListener()
  • Más flexible y moderno

removeEventListener()

×

¿Qué es removeEventListener()?

Método para remover un manejador de eventos previamente agregado con addEventListener().

Sintaxis:

elemento.removeEventListener(tipo, función);

Ejemplo:

const boton = document.getElementById('miBoton');

// Función de manejo
function manejarClick(e) {
    console.log('¡Clic!', e);
}

// Agregar evento
boton.addEventListener('click', manejarClick);

// Remover evento (debe ser la misma función)
boton.removeEventListener('click', manejarClick);

// Con función anónima (NO se puede remover)
boton.addEventListener('click', function() {
    console.log('No se puede remover');
});

Importante:

  • Debe ser exactamente la misma función
  • No funciona con funciones anónimas
  • Útil para limpiar eventos
  • Mejora el rendimiento

×

¿Qué es preventDefault()?

Previene el comportamiento por defecto de un evento del navegador.

evento.preventDefault();

// Prevenir envío de formulario

¿Qué es innerHTML?

Propiedad que permite obtener o establecer el contenido HTML completo de un elemento.

Sintaxis:

elemento.innerHTML = '<html>';
let contenido = elemento.innerHTML;

const div = document.getElementById('miDiv');

  • Reemplaza todo el contenido

textContent

×

¿Qué es textContent?

Propiedad que permite obtener o establecer solo el texto de un elemento, sin etiquetas HTML.

Sintaxis:

elemento.textContent = 'texto';
let texto = elemento.textContent;

Ejemplo:

const div = document.getElementById('miDiv');

// Establecer solo texto
div.textContent = 'Este es un texto simple';

// Obtener solo texto
console.log(div.textContent); // "Este es un texto simple"

// No interpreta HTML
div.textContent = '<strong>Texto</strong>';
// Resultado: "<strong>Texto</strong>" (literal)

Características:

  • Solo texto, sin HTML
  • No ejecuta scripts
  • Más seguro que innerHTML
  • Incluye texto de elementos hijos

innerText

×

¿Qué es innerText?

Propiedad que obtiene o establece el texto visible de un elemento, considerando los estilos CSS.

Sintaxis:

elemento.innerText = 'texto';
let texto = elemento.innerText;

Ejemplo:

const div = document.getElementById('miDiv');
div.innerHTML = '<p>Párrafo 1</p><p style="display:none">Oculto</p><p>Párrafo 2</p>';

console.log(div.textContent); // "Párrafo 1OcultoPárrafo 2"
console.log(div.innerText);   // "Párrafo 1Párrafo 2" (sin el oculto)

// Establecer texto
div.innerText = 'Nuevo texto visible';

Diferencias:

  • textContent: Todo el texto, sin considerar estilos
  • innerText: Solo texto visible, considera CSS
  • innerHTML: Incluye etiquetas HTML
  • innerText es más lento que textContent

Objeto que proporciona métodos para manipular las clases CSS de un elemento de forma más eficiente.
