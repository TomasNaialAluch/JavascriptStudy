Variables y Tipos de Datos

let nombre = "María";
let edad = 25;
let activo = true;
nombre = "María"
edad = 25
activo = true
String: Cadena de texto
Number: Valor numérico
Boolean: Verdadero/Falso

Explicación

Las variables son como cajas que guardan información. Cada tipo de dato tiene un propósito específico:

  • String: Para texto (nombres, mensajes)
  • Number: Para números (edades, precios)
  • Boolean: Para valores lógicos (verdadero/falso)

Callbacks - Funciones como Parámetros

function saludar(nombre) {
console.log(`Hola, ${nombre}!`);
}
function procesarEntrada(callback) {
let nombre = "María";
callback(nombre);
}
procesarEntrada(saludar);
saludar()
procesarEntrada()
nombre = "María"
Callback: función pasada como parámetro

Explicación del Callback

Un callback es una función que se pasa como parámetro a otra función. En este ejemplo:

  1. procesarEntrada() recibe una función como parámetro
  2. Crea una variable nombre = "María"
  3. Llama a la función callback pasándole el nombre
  4. saludar() se ejecuta e imprime el mensaje

Bucles For - Repetición Controlada

for (let i = 0; i < 3; i++) {
console.log(`Iteración ${i}`);
}
i = 0
console.log()
Iteración 0, Iteración 1, Iteración 2

Explicación del Bucle For

El bucle for repite un bloque de código un número específico de veces:

  1. Inicialización: let i = 0 (empezar en 0)
  2. Condición: i < 3 (mientras i sea menor que 3)
  3. Incremento: i++ (aumentar i en 1 cada vez)
  4. Ejecución: El código dentro del bucle se ejecuta
