Explicación de Selección DOM

JavaScript puede seleccionar elementos HTML de varias formas:

  • getElementById(): Selecciona por ID único
  • getElementsByClassName(): Selecciona por clase (devuelve colección)
  • querySelector(): Selecciona por selector CSS (más flexible)
  • querySelectorAll(): Selecciona múltiples elementos

Explicación de Event Bubbling

El event bubbling es el proceso por el cual los eventos se propagan hacia arriba en el DOM:

  • Fase 1: El evento se dispara en el elemento objetivo (botón)
  • Fase 2: El evento "burbujea" hacia el elemento padre (div hijo)
  • Fase 3: Continúa hacia el abuelo (div padre)
  • Control: event.stopPropagation() detiene la propagación

Manipulación del DOM - Crear y Modificar

// Crear elemento
let nuevoDiv = document.createElement('div');
nuevoDiv.textContent = 'Nuevo elemento';
// Agregar al DOM
document.body.appendChild(nuevoDiv);
// Modificar estilos
nuevoDiv.style.color = 'red';

Explicación de Manipulación DOM

JavaScript puede crear, modificar y eliminar elementos del DOM:

  • createElement(): Crea un nuevo elemento HTML
  • appendChild(): Agrega el elemento al DOM
  • textContent: Establece el contenido de texto
  • style: Modifica los estilos CSS del elemento
