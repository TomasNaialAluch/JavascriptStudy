Módulos ES6 en Acción

Visualiza cómo los módulos se comunican entre sí

Exportar Funciones y Variables

// utils.js
export const PI = 3.14159;
export function sumar(a, b) { return a + b; }
export function multiplicar(a, b) { return a * b; }
utils.js

Módulo Utils
PI

¿Qué está pasando?

El módulo utils.js está exportando sus funciones y variables para que otros módulos puedan usarlas. Cada export hace que ese elemento esté disponible para importar.

Importar desde Otros Módulos

// main.js
import { PI, sumar, multiplicar } from './utils.js';
console.log(PI); // 3.14159
console.log(sumar(2, 3)); // 5

¿Qué está pasando?

Exportación por Defecto

// Calculadora.js
export default class Calculadora {
sumar(a, b) { return a + b; }
restar(a, b) { return a - b; }
}

¿Qué está pasando?

export default permite exportar un solo elemento principal del módulo. Se importa sin llaves {} y puede tener cualquier nombre en el módulo que lo importa.
