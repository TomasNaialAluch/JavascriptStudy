Métodos de Array - Transformar Datos

let numeros = [1, 2, 3, 4, 5];
// map() - transformar cada elemento
let dobles = numeros.map(n => n * 2);
// filter() - filtrar elementos
let pares = numeros.filter(n => n % 2 === 0);

Explicación de map()

El método map() crea un nuevo array aplicando una función a cada elemento:

  • Entrada: Array original [1, 2, 3, 4, 5]
  • Función: n => n * 2 (multiplicar por 2)
  • Salida: Nuevo array [2, 4, 6, 8, 10]
  • No modifica: El array original permanece igual

Objetos - Estructura de Datos

let persona = {
nombre: "Ana",
edad: 25,
ciudad: "Madrid"
};
// Acceder a propiedades
console.log(persona.nombre); // "Ana"
Objeto persona
nombre: "Ana"
edad: 25
ciudad: "Madrid"
persona.nombre
Resultado: "Ana"

Explicación de Objetos

Los objetos almacenan datos en pares clave-valor:

  • Propiedades: nombre, edad, ciudad
  • Valores: "Ana", 25, "Madrid"
  • Acceso: persona.nombre o persona["nombre"]
  • Estructura: { clave: valor, clave: valor }

Destructuring - Extraer Valores

let persona = { nombre: "Carlos", edad: 30 };
// Destructuring tradicional
let nombre = persona.nombre;
let edad = persona.edad;
// Destructuring moderno
let { nombre, edad } = persona;
Objeto Original
nombre: "Carlos"
edad: 30
Variables Extraídas
nombre = "Carlos"
edad = 30
Destructuring: extrae propiedades automáticamente

Explicación de Destructuring

El destructuring permite extraer valores de objetos y arrays de forma concisa:

  • Tradicional: let nombre = persona.nombre;
  • Moderno: let { nombre, edad } = persona;
  • Ventajas: Más corto, más legible, menos repetición
  • Uso: Ideal para extraer múltiples propiedades
