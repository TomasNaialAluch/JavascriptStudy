# 🚀 JavaScript Junior Developer - Guía Completa

Una página web interactiva y moderna que sirve como guía completa para desarrolladores JavaScript Junior. Incluye conocimientos esenciales, herramientas, proyectos recomendados y oportunidades de trabajo reales.

## ✨ Características

### 🎯 Navegación Intuitiva
- **Header fijo** con navegación suave entre secciones
- **Sistema de breadcrumbs** dinámico
- **Menú hamburguesa** para dispositivos móviles
- **Scroll suave** y animaciones

### 💼 Oportunidades de Trabajo
- **3 ofertas realistas** con diferentes empresas mexicanas
- **Botones interactivos** para aplicar y guardar trabajos
- **Sistema de notificaciones** para feedback del usuario
- **Persistencia de datos** con localStorage

### 📚 Contenido Educativo
- **Conocimientos esenciales** de JavaScript
- **Herramientas de desarrollo** recomendadas
- **Proyectos prácticos** con diferentes niveles de dificultad
- **Información detallada** sobre cada tema

## 🛠️ Tecnologías Utilizadas

- **HTML5** - Estructura semántica
- **CSS3** - Estilos modernos y responsivos
- **JavaScript ES6+** - Funcionalidad interactiva
- **Font Awesome** - Iconos

## 🚀 Cómo Usar

1. **Clona el repositorio:**
   ```bash
   git clone https://github.com/tu-usuario/javascript-junior-developer.git
   ```

2. **Abre el archivo `index.html` en tu navegador:**
   - Simplemente haz doble clic en el archivo
   - O usa un servidor local como Live Server

3. **Explora las diferentes secciones:**
   - Navega usando el menú superior
   - Haz clic en los botones de oportunidades
   - Explora los proyectos recomendados
